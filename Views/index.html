An easier way to share files and links between devices!

There has always been an issue between the connectivity of different devices, therefore this website was created to help bridge the gap from different operating systems while remaining secure using a bidirectional peer-to-peer connection.

Peer-to-Peer Connection

Other than the initial connection through the server, all connections are securely linked using a peer-to-peer structure. Only the peers' browsers will share information.

Send Files and Messages

The host is able to transmit both files and messages to the client in a one-way fashion. Data sent is protected by the Hypertext Transfer Protocol Secure (HTTPS) and login system. WebRTC is end-to-end encrypted as well.

How to Use

1
Register or login into the system
2
After Logging in, connect to the peer of your choice by using their peer ID
3
If a single account transfer is needed, generate a new peer on the receiving end and enter the new peer id into the connect box.
